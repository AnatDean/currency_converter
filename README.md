# Currency Converter

## 1. Clone and navigate into repository

```bash
git clone https://github.com/AnatDean/currency_converter
cd currency_converter
```

## 2. Install dependencies (and wait a bit â°)

```bash
yarn ~ or ~ npm install
```

## 3. Start up

```bash

```
